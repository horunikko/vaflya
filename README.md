## Что из себя представляет Вафля?

Вафля — многофункциональный Telegram-бот, предназначенный для продажи VPN подписок. В первую очередь, Вафля направлен на простоту и минимализм, интуитивно понятные для пользвателя и в то же время простые для использования VPN провайдером.
